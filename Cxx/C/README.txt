
MacOS：
1.安装brew
    /bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
2.安装CMake
    brew install cmake
    cmake --version
3.安装docker（optional
    brew install docker
4.安装XCode开发工具（为了后面的gdb安装）
    xcode-select --install
5.安装gdb
    brew install gdb
6.给gdb签名
    https://blog.csdn.net/qq_33154343/article/details/104784641
